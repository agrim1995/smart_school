Academics

The Academics module serves to organize class timetables, assign teachers, manage subjects and subject groups, and facilitate student promotions for the next session. Here's a structured guide on how to perform various tasks within this module:

  1. Adding Section:Navigate to Academics > Sections. Enter the Section Name and click on Save. Edit or delete sections as needed.
  2. Adding Class:Go to Academics > Class. Enter the Class and select the associated Sections. Save the entry. Edit or delete classes as required.
  3. Assigning Class Teacher:Access Academics > Assign Class Teacher. Choose the Class, section, and Class Teacher(s). Save the assignment. Manage assigned class teachers through edit or delete options.
  4. Adding Subjects:Visit Academics > Subjects. Enter Subject Name, Subject Code, and select Subject Type (Theory/Practical). Save the subject entry. Edit or delete subjects as necessary.
  5. Adding Subject Group:Proceed to Academic > Subject Group. Input subject group Name, select Class, Section, Subjects, and provide a Description. Save the subject group. Edit or delete subject groups as needed.
  6. Adding Class Timetable:Go to Academic > Class Timetable. Click on Add and specify Class, Section, and Subject group. Use parameters to generate timetables quickly. Click +Add New to include multiple entries.
  7. Viewing Teacher Timetable:Navigate to Academics > Teacher Timetable. Select Teachers and click Search to view their timetables.
  8. Promoting Students:Access Academics > Promote Student. Select Class and Section, then click Search. Review current results and next session status. Choose students for promotion by selecting Promote In Session, Class, and Section. Click Promote to execute the promotion process.
  9. Student Access to Class Timetable:In the student panel, navigate to the sidebar menu and click on Class Timetable. View the weekly class timetable.
  10. Promoting Students for Next Session:Go to Academics > Promote Students. Select the Class and Section. Specify promotional details and click Search. Review the list of students and select those to be promoted. Click Promote to finalize the process. This structured guide facilitates efficient management of academic processes, ensuring smooth functioning of class schedules, teacher assignments, subject management, and student promotions.
